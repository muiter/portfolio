Insightful Quotes comes from setting up a MySQL database to store quotes inserted by various users. I create an AJAX request to a PHP page to retrieve all quotes stored in the database and then store them in an array. By randomly choosing an index out of the array, I show random quotes to users upon the pages initial load. The refresh icon in the upper right hand corner shows the next quote in the array.
